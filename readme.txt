== Installation ==
1. Upload `viralparse.zip` to your computer.
2. Go to your WordPress Dashboard and select Plugins >> Add New.
3. Click on the Upload option at the top and select the `viralparse.zip file` you just downloaded.
4. Click on Install.
5. Activate the plugin through the \'Plugins\' menu in WordPress
6. There should appear Dashboard between your \"Dashboard\" menu and \"Posts\".
7. Clicking on \"ViralParse\" and start follow the steps to connect it with ViralParse.Com
